EUDI Verify

Open-source EU Digital Identity verification. Drop the <eudi-verify> widget into any website. Your server validates tokens — the browser never holds trusted identity data.

Try Age Verification Demo

EU Digital Identity Wallets launch December 2026. This site runs in demo mode (simulated wallet) — no real credentials involved. Integrators can point the same stack at a real wallet locally; see docs/SUPPORTED.md and examples/server.

How it works

  1. 1
    Scan QR code

    Open the built-in demo wallet — or, in a local lab setup, scan with a real EUDI Wallet app.

  2. 2
    Approve in wallet

    User reviews requested claims and approves the presentation.

  3. 3
    Server verification

    Your backend validates the HMAC token — no client-side trust required.
