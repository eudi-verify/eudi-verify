How it works

  1. 1
    Scan QR code

    User scans with their EUDI Wallet app — or opens the built-in demo wallet.

  2. 2
    Approve in wallet

    User reviews requested claims and approves the presentation.

  3. 3
    Server verification

    Your backend validates the HMAC token — no client-side trust required.
