# Support the project

`eudi-verify` is independent open source (Apache-2.0). Core packages stay free and self-hostable.

## Contribute without donating

- Try the [html-vanilla demo](examples/html-vanilla/) and [open issues](https://github.com/eudi-verify/eudi-verify/issues) for bugs or gaps
- Send pull requests — see [CONTRIBUTING.md](CONTRIBUTING.md) and [docs/SUPPORTED.md](docs/SUPPORTED.md)
- Run interop tests against your wallet or sandbox and share results

## Sponsor

GitHub Sponsors and Open Collective are **not configured yet**. When they are, links will appear on the repo **Sponsor** button via [`.github/FUNDING.yml`](.github/FUNDING.yml).

Sponsorship helps pay for:

- Security review and accessibility work on `<eudi-verify>`
- Adoption documentation and a hosted demo
- Maintainer time for releases and ecosystem interop

**License unchanged:** sponsorship does not affect the Apache-2.0 license on core packages.

## Enterprise

For procurement, support contracts, or other commercial inquiries, open a [GitHub Discussion](https://github.com/eudi-verify/eudi-verify/discussions) with subject **Enterprise inquiry**.

## Acknowledgements

When we receive significant public funding or sponsorship, we will note it here and in release notes.
